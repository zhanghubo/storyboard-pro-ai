# Storyboard Grid Pro：智能分镜网格生成器

专业的分镜规划工具，支持9/15/20个分镜的智能网格布局和AI提示词生成。

## 🎬 核心特性

- **智能网格布局**：支持3x3、3x5、4x5三种专业网格配置
- **自动分镜分配**：基于叙事结构的智能算法
- **可视化编辑器**：直观的拖拽式分镜规划界面
- **提示词生成引擎**：自动生成适配AI绘画的专业提示词
- **多格式导出**：支持JSON、PDF、PNG等多种导出格式
- **视觉流优化**：智能优化分镜间的视觉焦点移动

## 📦 快速开始

### 安装依赖

```bash
npm install
```

### 运行开发服务器

```bash
npm run dev
```

在浏览器中打开 [http://localhost:3000](http://localhost:3000)

## 🎯 使用场景

- **短视频制作**：快速规划短视频分镜（9分镜）
- **广告创意**：标准叙事结构（15分镜）
- **电影场景**：史诗级长镜头序列（20分镜）

## 📐 网格配置

| 分镜数 | 布局 | 纵横比 | 适用场景 |
|--------|------|--------|----------|
| 9个 | 3x3 | 16:9 | 快速叙事、广告 |
| 15个 | 3x5 | 16:9 | 标准叙事、预告片 |
| 20个 | 4x5 | 16:9 | 史诗叙事、电影场景 |

## 🛠️ 技术栈

- **前端**：Next.js + React
- **动画**：Framer Motion
- **图标**：Lucide React
- **导出**：jsPDF + html2canvas
