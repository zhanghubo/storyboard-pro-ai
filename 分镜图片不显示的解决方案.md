# 🖼️ 分镜图片不显示的解决方案

## 📊 当前情况分析

从您的截图看到：
- ✅ 分镜网格显示正常（15个格子）
- ✅ 文字描述显示正常（镜头类型、时长等）
- ❌ **图片没有显示在格子背景中**

---

## 🔍 问题原因

### 最可能的原因：没有启用图像生成

您在配置时**没有勾选"✅ 启用图像生成"**，所以系统只生成了文字描述，没有调用图像API生成图片。

---

## ✅ 解决方案（3步）

### 步骤1：检查配置

**查看页面顶部的提示**：

如果您看到类似这样的黄色提示框：
```
⚠️ 提示：仅生成了文本提示词

当前系统只生成了分镜描述和AI绘画提示词。

想要生成图片和视频？请按以下步骤操作：
1. 点击下方"🔄 重新开始"按钮
2. 在第一步"⚙️ 完整AI配置"中
   ✅ 勾选"启用图像生成"
   ...
```

**这就证实了：您没有启用图像生成功能。**

---

### 步骤2：重新配置并启用图像生成

#### A. 点击"重新开始"
在结果页面底部，点击**"🔄 重新开始"**按钮

#### B. 重新配置API

在第一步"⚙️ 完整AI配置"页面：

1. **文本生成配置**（必需）
   ```
   ✅ 已配置（保持不变）
   ```

2. **图像生成配置**（重要！）
   ```
   ✅ 勾选"启用图像生成" ← 这个必须勾选！
   
   选择提供商：Doubao Image（推荐）
   API密钥：填入您的API密钥
   Endpoint ID：填入图像模型的Endpoint
   ```

3. **视频合成配置**（可选）
   ```
   □ 暂时不勾选（可以后面再试）
   ```

#### C. 继续配置
点击**"开始配置"**按钮

---

### 步骤3：重新生成分镜

1. **选择分镜数量**：9个或15个
2. **输入故事内容**：（和之前一样）
3. **选择视觉风格**：电影感
4. **点击"生成分镜"**

**等待时间**：
- 文字生成：10-30秒
- 图片生成：1-3分钟（15张图）

## ⚠️ 常见问题

### Q1: 我确实勾选了，但还是没有图片？

**诊断步骤**：

1. **按F12打开浏览器控制台**
2. **切换到Console标签**
3. **查找关键信息**：
   ```
   搜索：[图像生成]
   或：正在生成分镜图片
   ```
4. **查看是否有错误**（红色文字）

**可能的错误**：
- API密钥错误 → 重新获取密钥
- Endpoint ID错误 → 确认是图像模型的Endpoint
- 额度不足 → 充值
- 网络超时 → 重试

---

### Q2: 生成图片需要多长时间？

**时间参考**：
- 9个分镜：约1-2分钟
- 15个分镜：约2-3分钟
- 20个分镜：约3-5分钟

**请耐心等待，不要刷新页面！**

---

### Q3: 部分图片生成失败怎么办？

系统会尽力生成所有图片，但如果某些失败：
- ✅ 成功的图片会显示
- ❌ 失败的格子只显示文字
- 💡 可以单独重新生成失败的分镜

---

### Q4: 如何获取图像模型的Endpoint ID？

#### 火山引擎豆包图像模型：

1. 访问：https://console.volcengine.com/ark
2. 进入**"模型推理"** → **"在线推理"**
3. 选择**"图像生成"**分类
4. 找到图像模型（如"豆包图像生成"）
5. 点击**"接入点管理"**
6. 创建或查看接入点
7. 复制 **Endpoint ID**（格式：`ep-xxxxxxxx-xxxx`）
