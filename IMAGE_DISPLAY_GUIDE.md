## 🎨 配置图像生成的步骤

### 步骤1：配置图像API
在"⚙️ 完整AI配置"的第二步：
1. ✅ 确保"**启用图像生成**"已勾选（默认已勾选）
2. 填入火山引擎**图像模型**的 Endpoint ID
3. 填入 API 密钥

### 步骤2：生成分镜
- 输入故事内容
- 点击"开始生成"
- 等待图片生成（2-10分钟）

### 步骤3：查看结果
系统会自动：
- ✅ 在网格中显示图片
- ✅ 在画廊中展示图片
- ✅ 提供下载功能

## 🔧 故障排除

### Q1: 网格中没有显示图片？
**检查项：**
- ✅ 是否勾选了"启用图像生成"
- ✅ 是否填写了图像模型的 Endpoint ID
- ✅ 图像是否生成成功（查看图片画廊区域）
- ✅ 刷新页面重试

### Q2: 图片很模糊？
**原因：** 网格中的图片是缩略图
**解决：**
- 点击网格查看原图
- 在图片画廊中查看完整图片
- 下载后在本地查看

### Q3: 只看到部分图片？
**可能原因：**
- 部分图片生成失败
- 网络问题导致加载失败
- API限流或配额不足

**解决方法：**
- 检查浏览器控制台错误
- 重新生成失败的分镜
- 检查API配额和余额

### Q4: 图片显示位置不对？
**确认：** 图片编号是否与分镜编号对应
- 第1张图片 → 分镜#1
- 第2张图片 → 分镜#2
- ...以此类推
