# 🎬 Storyboard Grid Pro AI

**智能分镜网格生成器 - 完整的AI驱动视频制作工作流**

从故事文本到最终视频，一站式AI创作平台

## ✨ 核心特性

### 🤖 三合一AI工作流

```
📝 文本输入 → 🎨 AI分镜描述 → 🖼️ 图像生成 → 🎬 视频合成
```

1. **文本生成** - 大语言模型生成专业分镜描述
2. **图像生成** - 文生图模型为每个分镜创建图片
3. **视频合成** - 智能合成最终故事板视频

### 🎯 完整功能

- ✅ **多种分镜布局** - 9/15/20个分镜网格
- ✅ **智能场景分配** - 自动优化分镜视觉流
- ✅ **AI提示词生成** - 专业级图像生成提示词
- ✅ **批量图像生成** - 自动生成所有分镜图片
- ✅ **视频自动合成** - 添加转场效果的完整视频
- ✅ **多格式导出** - MD/DOC/PDF/HTML/JSON/TXT
- ✅ **实时进度追踪** - 可视化生成进度
- ✅ **灵活配置** - 可选择性启用各个AI功能

## 🚀 快速开始

### 启动服务器

```bash
npm run dev
```

访问 **http://localhost:3000**

### 三步开始创作

1. **配置API** - 填入火山引擎豆包的Endpoint ID和API密钥
2. **输入故事** - 描述您的故事情节
3. **生成作品** - 等待AI自动完成全部工作

## 🎬 工作流程

### 步骤1：生成分镜描述（10-30秒）
AI将故事拆分成多个场景，生成详细描述。

### 步骤2：创建提示词（5秒）
为每个分镜生成专业的AI绘画提示词。

### 步骤3：生成图片（2-10分钟，可选）
使用文生图API为每个分镜创建图片。

### 步骤4：合成视频（1-5分钟，可选）
将所有图片合成为带转场效果的视频。

## 🏗️ 技术栈

- **Next.js 14** + TypeScript
- **Tailwind CSS** + Framer Motion
- 多AI服务商集成
- 完整导出功能

- **图片：** 1920×1080 (16:9) 或 1080×1920 (9:16)
- **视频：** MP4, H.264, 30fps
- **文档：** MD/DOC/HTML/JSON/TXT
