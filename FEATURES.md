# Storyboard Grid Pro - 功能说明

## 🎯 核心功能

### 1. AI模型配置 ⚙️
支持6大主流AI服务商：
- 🤖 **OpenAI** - GPT-4, GPT-4-turbo, GPT-3.5-turbo
- 🔍 **DeepSeek** - deepseek-chat, deepseek-coder
- 🧠 **Anthropic Claude** - Claude-3系列
- 🌋 **火山引擎豆包** - doubao-pro-4k, doubao-lite-4k
- ☁️ **通义千问** - qwen-max, qwen-plus, qwen-turbo
- ⚡ **智谱AI** - glm-4, glm-3-turbo

**API连接测试指示灯**：
- 🔴 灰色 - 未测试
- 🟡 黄色闪烁 - 测试中
- 🟢 绿色发光 - 连接成功
- 🔴 红色 - 连接失败

### 2. 网格布局选择 📐
三种专业分镜配置：
- **9个分镜** (3x3) - 快速叙事，适合短视频、广告
- **15个分镜** (3x5) - 标准叙事，适合短片、预告片
- **20个分镜** (4x5) - 史诗叙事，适合长视频、电影场景

### 3. 输出模式配置 📤

#### 单图模式 🎴
- 生成 **N张独立的分镜图片**（N = 9/15/20）
- 每张图片独立输出，可单独使用
- 适合：
  - 需要高质量单张输出
  - 分别用于不同AI绘画工具
  - 需要单独编辑每个分镜

#### 拼图模式 🎬
- 生成 **1张包含所有分镜的大图**
- 所有分镜按网格排列在一张图中
- 适合：
  - 预览完整故事板
  - 打印输出
  - 快速展示整体效果

### 4. 画面比例选择 📐

#### 横版 16:9 ⬜
- 尺寸：1920 × 1080
- 适合：
  - 电脑屏幕观看
  - 横屏视频
  - 传统电影格式

#### 竖版 9:16 📱
- 尺寸：1080 × 1920
- 适合：
  - 手机竖屏观看
  - 短视频平台（抖音、快手）
  - 社交媒体Story

### 5. 视觉风格预设 🎨
- 🎬 **电影感** - 好莱坞风格，戏剧性光影
- 🎨 **动画风格** - 卡通/动画，鲜艳色彩
- 📹 **纪实风格** - 真实自然，纪录片质感
- 🚀 **科幻风格** - 未来感，科技氛围
- 🔮 **奇幻风格** - 魔法神秘，史诗奇幻

### 6. 智能分镜生成 🤖
- AI自动分析故事结构
- 智能分配镜头类型（特写、广角、中景等）
- 优化视觉流动路径
- 生成专业AI绘画提示词

### 7. 多格式导出下载 💾
- 📝 **Markdown** (.md) - 适合编辑和版本控制
- 📄 **Word文档** (.doc) - 可在Word中打开编辑
- 🌐 **网页格式** (.html) - 可在浏览器打开，支持打印
- 🔧 **JSON数据** (.json) - 供程序处理
- 📃 **纯文本** (.txt) - 通用格式

## 🎯 使用场景示例

### 场景1：短视频制作（抖音/快手）
```
配置：
- 分镜数量：9个（3x3）
- 输出模式：单图模式
- 画面比例：9:16（竖版）
- 视觉风格：动画风格

结果：生成9张独立的竖版分镜图片
用途：每张图片单独用于AI绘画，生成竖屏短视频素材
```

### 场景2：电影场景预览
```
配置：
- 分镜数量：20个（4x5）
- 输出模式：拼图模式
- 画面比例：16:9（横版）
- 视觉风格：电影感

结果：生成1张包含20个分镜的横版大图
用途：打印输出或展示完整故事板
```

### 场景3：广告创意提案
```
配置：
- 分镜数量：15个（3x5）
- 输出模式：单图模式
- 画面比例：16:9（横版）
- 视觉风格：纪实风格

结果：生成15张独立的横版分镜图片
用途：分别生成高质量图片，用于广告提案
```

## 🎨 提示词格式

每个分镜生成的提示词包含：
1. **镜头技术规格** - 镜头类型、网格位置
2. **场景描述** - 详细的视觉内容
3. **风格标签** - 光线、色彩、情绪
4. **纵横比参数** - --ar 16:9 或 --ar 9:16

示例：
```
[Shot 1/15: Extreme wide establishing shot] [Grid Position: A1] [Key Frame]
黎明时分的南太平洋无人岛全景，无人机视角从海面升起，展示郁郁葱葱的丛林和中央的山峰，
海上薄雾缭绕，电影感，史诗感，黄金时刻光线，超详细渲染，cinematic, professional photography, 
high detail, masterpiece, ultra detailed, 8k resolution, dramatic lighting
--ar 16:9 --style raw
```

## 🚀 快速开始

1. **配置AI** → 选择服务商，输入API密钥，测试连接（看到绿灯）
2. **选择网格** → 选择9/15/20个分镜
3. **配置输出** → 选择单图/拼图模式，选择16:9/9:16比例
4. **输入故事** → 描述故事情节，选择视觉风格
5. **生成分镜** → AI自动生成专业分镜和提示词
6. **下载导出** → 选择需要的格式下载

## 💡 最佳实践

1. **单图模式** 适合需要高质量独立输出的场景
2. **拼图模式** 适合快速预览和打印展示
3. **竖版9:16** 专为移动端和短视频平台优化
4. **横版16:9** 适合传统视频和电脑观看
5. 详细的故事描述能生成更精准的分镜
6. 测试API连接后再开始，确保生成顺利
