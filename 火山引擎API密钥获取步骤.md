# 🔑 火山引擎API密钥获取步骤

## ⚠️ 您遇到了401错误：API密钥格式不正确

### 正确获取API密钥的步骤：

---

## 第1步：访问火山引擎控制台

**网址**：https://console.volcengine.com/ark

---

## 第2步：获取API密钥

### 方法A：从右上角获取（推荐）

1. **点击页面右上角的"用户名"或"头像"**
2. **在下拉菜单中选择"API访问密钥"**
3. **在密钥管理页面，找到已有的密钥或创建新密钥**
4. **点击"查看"或"复制"按钮**
5. **完整复制整个密钥（应该是一长串字符，以 `AKLT` 开头）**

### 方法B：从密钥管理页面

1. **访问**：https://console.volcengine.com/iam/keymanage/
2. **找到"访问密钥（Access Key）"**
3. **复制完整的 Access Key ID（以 AKLT 开头）**

---

## 第3步：验证密钥格式

### ✅ 正确的密钥格式

```
AKLT开头的长字符串
例如：AKLTxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
长度：约40-50个字符
```

### ❌ 错误的密钥格式

```
❌ 不以AKLT开头
❌ 太短（少于30个字符）
❌ 包含空格或换行
❌ 只复制了一部分
❌ 使用了Secret Access Key（这个是密钥，不是密钥ID）
```

---

## 第4步：重新配置

### 在浏览器中：

1. **访问**：http://localhost:3000

2. **如果已经在结果页面**：
   - 点击"🔄 重新开始"按钮
   - 返回到配置页面

3. **在配置页面**：
   - 找到"API密钥"输入框
   - **删除旧的密钥**
   - **粘贴新复制的完整密钥**
   - ⚠️ **确保没有多余的空格或换行**

4. **填写Endpoint ID**：
   ```
   文本模型：ep-xxxxxxxx-xxxx
   图像模型：ep-xxxxxxxx-xxxx
   视频模型：ep-xxxxxxxx-xxxx
   ```

5. **点击"🔌 测试连接"按钮**
   - 等待测试结果
   - 看到 **🟢 "✓ 连接成功"** 即可

6. **点击"开始配置"继续**

---

## 📌 重要提示

### 🔐 API密钥 vs Secret Key

火山引擎有两个密钥：

| 名称 | 用途 | 格式 | 是否使用 |
|------|------|------|---------|
| **Access Key ID** | API密钥 | `AKLT...` | ✅ **使用这个** |
| **Secret Access Key** | 密钥 | 随机字符 | ❌ 不需要 |

### ⚠️ 您只需要 Access Key ID（以AKLT开头的那个）

---

## 🔍 如何验证密钥是否正确

### 检查清单：

- [ ] 密钥以 `AKLT` 开头
- [ ] 密钥长度约 40-50 个字符
- [ ] 没有空格
- [ ] 没有换行符
- [ ] 是从"API访问密钥"页面获取的
- [ ] 是 Access Key ID，不是 Secret Key

---

## 💡 如果还是不行

### 尝试创建新的密钥

1. **访问**：https://console.volcengine.com/iam/keymanage/
2. **点击"新建密钥"按钮**
3. **系统会生成一个新的 Access Key**
4. **立即复制新的 Access Key ID**
5. **使用这个新密钥重新配置**

---

## 📞 获取帮助

如果按照以上步骤操作后仍然出现错误：

1. **检查终端日志**：
   - 查看是否有更详细的错误信息
   - 记录错误的Request ID

2. **查看浏览器控制台**：
   - 按F12打开开发者工具
   - 切换到"Console"标签
   - 查看错误详情

3. **联系火山引擎技术支持**：
   - 提供错误截图
   - 提供Request ID
   - 说明您的账号和问题
