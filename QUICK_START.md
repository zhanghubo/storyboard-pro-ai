# 🚀 快速开始指南

## ⚠️ 重要：生成图片和视频必须配置API

如果您只看到**文本提示词**而没有看到**图片**和**视频**，说明您没有配置图像和视频API！

---

## 📋 三步快速配置

### 步骤1️⃣：配置文本生成API（必需）

**这一步用于生成分镜描述和提示词**

1. 访问 [火山引擎控制台](https://console.volcengine.com/ark)
2. 选择大语言模型（如 Doubao-pro-4k）
3. 创建推理接入点
4. 复制 **Endpoint ID**（格式：`ep-xxxxx-xxxxx`）
5. 复制 **API Key**

在配置页面填入：
- ✅ Endpoint ID：`ep-xxxxx-xxxxx`
- ✅ API密钥：`your-api-key`

---

### 步骤2️⃣：配置图像生成API（生成图片必需）⭐

**没有这一步，就不会生成图片！**

1. 在火山引擎控制台进入"**图像生成**"模块
2. 选择文生图模型
3. 创建推理接入点
4. 复制图像模型的 **Endpoint ID**

在配置页面填入：
- ✅ 勾选"**启用图像生成**"（默认已勾选）
- ✅ Endpoint ID：`ep-xxxxx-xxxxx`（图像模型的ID）
- ✅ API密钥：`your-api-key`（可以和文本API用同一个）

**重要提示：**
- 如果不填这个，系统只会生成文本提示词
- 填写后会为每个分镜自动生成图片

---

### 步骤3️⃣：配置视频生成API（可选）

**如果想要最终视频，需要这一步**

1. 在火山引擎控制台进入"**视频生成**"模块
2. 选择视频合成模型
3. 创建推理接入点
4. 复制视频模型的 **Endpoint ID**

在配置页面填入：
- ✅ 勾选"**启用视频合成**"
- ✅ Endpoint ID：`ep-xxxxx-xxxxx`（视频模型的ID）
- ✅ API密钥：`your-api-key`

## 📸 如何获取火山引擎 Endpoint ID

### 1. 登录控制台
访问：https://console.volcengine.com/ark

### 2. 选择模型类型

#### 文本模型（必需）
- 进入"模型推理"
- 选择 Doubao-pro-4k 或 Doubao-lite-4k
- 点击"创建推理接入点"
- 复制生成的 Endpoint ID

#### 图像模型（生成图片必需）
- 进入"**图像生成**"模块
- 选择文生图模型（如 stable-diffusion-xl）
- 点击"创建推理接入点"
- 复制生成的 Endpoint ID

#### 视频模型（可选）
- 进入"**视频生成**"模块
- 选择视频合成模型
- 点击"创建推理接入点"
- 复制生成的 Endpoint ID

### 3. 获取API密钥
- 在控制台右上角点击"API密钥"
- 创建或复制现有的API Key
- **所有三个模型可以使用同一个API Key**

## 🔧 常见问题

### Q1: 为什么我只看到提示词，没有图片？
**A:** 您没有配置图像生成API，或者没有勾选"启用图像生成"。

**解决方法：**
1. 点击"🔄 重新开始"
2. 在第一步配置中勾选"启用图像生成"
3. 填写图像模型的 Endpoint ID 和 API密钥

### Q2: 我配置了图像API，为什么还是没有图片？
**A:** 可能的原因：
1. Endpoint ID 填错了（应该是图像模型的ID，不是文本模型的）
2. API密钥无效或余额不足
3. 图像生成失败（查看浏览器控制台错误信息）

### Q3: 三个API可以用同一个密钥吗？
**A:** 可以！只要都是火山引擎豆包的服务，可以使用同一个API Key。

### Q4: 必须配置全部三个API吗？
**A:** 不必须。根据需求选择：
- 只要提示词：只配置文本API
- 要生成图片：配置文本 + 图像API
- 要生成视频：配置全部三个API

### Q5: 为什么图像生成这么慢？
**A:** 图像生成比文本慢很多：
- 每张图片需要 5-30秒
- 15个分镜大约需要 2-5分钟
- 这是正常现象，请耐心等待

### Q6: 费用会不会很贵？
**A:** 根据使用量：
- 仅提示词：几乎免费（¥0.04）
- 生成图片：适中（¥1-2）
- 生成视频：较贵（¥20+）

建议先测试少量分镜（9个），确认效果后再生成更多。
