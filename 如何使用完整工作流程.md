## ⚡ 60秒快速开始

### 步骤 1：打开配置页面（10秒）
```

```

### 步骤 2：填写三个API配置（30秒）

#### A. 文字大模型（必需）✅
```
✓ 选择提供商：Doubao（豆包）
✓ API密钥：AKLT开头的密钥
✓ Endpoint ID：doubao-seed-1-6-lite-251015
✓ 点击"测试连接" → 看到🟢成功

#### B. 图像生成（推荐）✅
```
✓ 勾选"✅ 启用图像生成"
✓ 选择：Doubao Image
✓ API密钥：同上
✓ Endpoint ID：你的图像模型Endpoint
```

#### C. 视频合成（可选）⏸️
```
□ 暂不勾选（可跳过）
```

1. 选择：15个分镜
2. 输入故事：一个女孩在海边看日落
3. 选择风格：电影感
4. 点击"生成分镜"
```

## 🛠️ 故障排查

### ❌ 问题：只看到提示词，没有图片

**原因**：未勾选"启用图像生成"

**解决**：
```
1. 点击页面底部"🔄 重新开始"按钮
2. 在第一步配置页面
3. 找到"图像生成（文生图模型）"区域
4. 勾选"✅ 启用图像生成"
5. 填写图像API配置
6. 重新生成
```

---

### ❌ 问题：图片生成失败

**可能原因**：
- API密钥错误 → 检查密钥是否正确
- Endpoint ID错误 → 确认是图像模型的Endpoint
- 额度不足 → 充值或更换API

**检查方法**：
```
查看浏览器控制台（F12）
查看终端日志
使用"测试连接"功能
```

---

### ❌ 问题：视频生成失败

**这是正常的！**

视频生成失败不影响图片：
- ✅ 图片已经生成
- ✅ 提示词已经生成
- ❌ 只有视频失败

**为什么？**
- 视频生成需要更长时间
- 视频API可能不稳定
- 成本较高，可以手动合成

## 💬 常见疑问

**Q: 必须配置三个API吗？**  
A: 不是。只需要文字大模型是必需的。图像和视频是可选的。

**Q: 不配置图像API会怎样？**  
A: 只会生成文字提示词，网格里不会有图片，但提示词可以导出到其他平台使用。

**Q: 图片生成很慢正常吗？**  
A: 正常。15张图片需要1-2分钟，请耐心等待。

**Q: 为什么视频总是失败？**  
A: 视频生成要求高、耗时长、成本高。建议先不配置视频，只生成图片即可。

**Q: 生成的图片在哪里？**  
A: 
1. 在结果页面的网格中（背景图）
2. 结果页面下方的"图片画廊"
3. 可以单独下载或批量下载
